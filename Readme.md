# Deimos Tech – Web Track Entry-Level Task

## Overview
This project contains the completed Profile Card component for the Deimos Tech Web Track Entry-Level Task. The provided starter code was updated to match the given design specification by fixing the listed UI issues.

## Changes Made
- Centered the profile card on the page.
- Ensured the avatar is displayed as a perfect circle.
- Centered the name and role text.
- Added the missing divider line.
- Updated the button color to `#533483`.
- Added a darker hover effect for the button.
- Improved spacing for a more consistent layout.
- Applied the required `16px` border radius to the card.

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone the repository.
2. Open `starter_profile_card.html` in any modern web browser.
